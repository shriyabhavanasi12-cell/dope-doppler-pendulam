# dope-doppler-pendulam
a simulation of doppler pendulam that i made for my physics assessment. this simulation shows the compression and expansion of waves when the pendulam is in motion ,thus highlight the doppler effect acting on a oscillatory motion of the  pendulam.it also contains feature sound where the  doppler  effect can be understood through change in the pitch of the sound. i made this using html ,inline css and javascript in VS code
